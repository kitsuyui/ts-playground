# ts-playground

[![codecov](https://codecov.io/gh/kitsuyui/ts-playground/branch/main/graph/badge.svg?token=E6RO7KERTY)](https://codecov.io/gh/kitsuyui/ts-playground)

Playground for TypeScript

## Usage

### Build

```sh
yarn build
```

### Run with development mode (watch mode)

```sh
yarn dev
```

## Test

```sh
yarn test
```
